Sort and Compare Algorithms

This is a project made as part of CSE 5311 Design and Analysis of Algorithms Fall'22 course taught at The University of Texas at Arlington

  • Perform Sorting
  • Compare Sorting Algorithms

Enter the input to perform sorting in the following format: <size_of_input> <algorithm>
Size of Input: Enter any integer number

Algorithm: Enter the number corresponding to the algorithm
1. Bubble Sort
2. Selection Sort
3. Insertion Sort
4. Merge Sort
5. Quick Sort (regular)
6. Quick Sort with 3 medians
7. Heap Sort
Example of sample input: 10 2
The [10 2] input will perform Selection Sort on a set of integers of size 10.

Enter the input to perform sorting in the following format: <size_of_input> <number_for_algorithms>
Size of Input: Enter any integer number
Algorithm: Enter the number corresponding to the algorithm to compare
1. Bubble Sort
2. Selection Sort
3. Insertion Sort
4. Merge Sort
5. Quick Sort (regular)
6. Quick Sort with 3 medians
7. Heap Sort
Example of sample input: 10 2 4
The [10 2 4] input will compare Selection Sort and Merge Sort for a set of integers of size 10.
